# Sistema de Gerenciamento de Monitoria

Este projeto é uma aplicação Web desenvolvida em **Python** para a disciplina **Python RAD**, ministrada pelo Prof.
[cite_start]Heleno Cardoso[cite: 50].

## Funcionalidades (Requisitos Funcionais)
[cite_start]Conforme solicitado no barema[cite: 35]:
1. [cite_start]**RF01 - Login**: Sistema de autenticação para acesso seguro[cite: 34].
2. **RF02 - Cadastro (CRUD)**: Interface para inclusão de novas monitorias no Banco de Dados.
3. **RF03 - Listagem**: Visualização em tempo real dos dados armazenados.
4. [cite_start]**RF04 - Relatório**: Gráficos estatísticos das monitorias por disciplina[cite: 36].

## 🛠️ Tecnologias Utilizadas
* [cite_start]**Linguagem**: Python 3.x.
* [cite_start]**Framework Web**: Streamlit (RAD)[cite: 32].
* [cite_start]**Banco de Dados**: SQLite (Relacional)[cite: 32].
* [cite_start]**Cloud**: Hospedado via Streamlit Cloud / Railway.
